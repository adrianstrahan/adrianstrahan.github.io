Wordygram

Wordygram is a simple anagram game where you find words from a set of letters presented to you in a wheel. In the middle of the wheel is a letter that must appear in each of your guesses. You can only use each letter once.

Collect points for each word you find and get bonus points for words using all the letters. Your correct guesses appear in a grid in alphabetical order so you can deduce any words you've missed.

When you've finished the game or seen the solution you can share your score with your friends.

You can buy extra games with in-app purchases.

Having problems?

If you have any problems with the game or require help with an in-app purchase, please contact us directly by emailing wordygram@usegis.co.uk. Please clearly state the problem you're having, and we'll get back to you as soon as possible.
