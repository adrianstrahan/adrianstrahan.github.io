Wordygram

Wordygram is a simple anagram game where you find the words from a set of letters presented to you in a wheel with a middle letter that must appear in each of your guesses. You can only use each letter once.

Collect points for each correct answer and get bonus points for words using all the letters. Your correct guesses appear in a grid in alphabetical order so you can deduce any words you've missed.

When you've finished the game or seen the solution you can share your score with your friends.

You can buy extra games with in-app purchases.
